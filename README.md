# virtual-icu-demo
Віртуальне відділення інтенсивної терапії: інтерактивна платформа на базі ШІ для прогнозування післяопераційних ускладнень у кардіохірургії. Впровадження Digital Twin пацієнта для підтримки клінічних рішень та навчання інтернів НМУ. Використовує ML-моделі для аналізу даних та симуляції критичних станів у реальному часі.
